    \large Программа для определения коэффициента подъемной силы летательного аппарата
\end{center}

\section*{Аннотация}
Программа реализована на языке Python с использованием библиотеки AeroBDSM для расчета аэродинамических характеристик. 
Выполняет расчет коэффициента подъемной силы с учетом интерференции между корпусом и несущими поверхностями, 
скоса потока, а также влияния числа Маха и угла атаки.

\section*{Описание основных модулей}

\subsection*{Класс ProbabilityIntegral}
Реализует таблицу значений интеграла вероятностей с линейной интерполяцией для промежуточных значений.

\subsection*{Глобальные параметры}
\begin{itemize}
    \item Геометрические характеристики летательного аппарата
    \item Параметры крыла и рулевых поверхностей  
    \item Коэффициенты интерференции
    \item Диапазоны чисел Маха и углов атаки для расчетов
\end{itemize}

\subsection*{Функция main()}
Выполняет последовательный расчет:
\begin{enumerate}
    \item Носовой части летательного аппарата
    \item Изолированного крыла
    \item Коэффициентов интерференции
    \item Скоса потока
    \item Суммарного коэффициента подъемной силы
\end{enumerate}
